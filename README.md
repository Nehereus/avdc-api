# AVDC

AVDC Powered by Flask

### Install

```shell
pip3 install -r requirements.txt
```

### Run

```shell
python3 main.py
```

### API

| Endpoint | Description |
| --- | ----------- |
| /image/`<id>` | Fetch `2:3` Poster Image |
| /metadata/`<id>` | Fetch Video Metadata |
| /people/`<name>` | Fetch Actress Avatar |
